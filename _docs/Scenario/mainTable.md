<h5>Общий табель</h5>

Цели ведения табеля учета рабочего времени:
--- Учитывать время , отработанное или не отработанное наемным работником
--- Контролировать соблюдения графика рабочего времени (явки, неявки, опоздания)
--- Иметь официальные сведения об отработанном каждым работником времени для расчета заработной платы  или статической отчетности

Пример официального документа табельного учета рабочего времени показан ниже:

<img src="/images/mainTable/officialTable.jpg" width="100%" height="auto"><br>

При разработке системы общего табеля учета рабочего времени необходимо учитывать следующие значения:
---ФИО - данное поле заполняется исходя из формы создания сотрудника
---Должность - заполняется исходя из формы создания сотрудника (учитывается привязанная к нему должность)
---Вывод всех чисел текущего месяца - данные поля заполняются указателем о явках и неявках (Я- явка, Н-неявка, О-отпуск, Б-больничный, К-командировка), а также количеством фактически отработанного времени сотрудником за текущий день.
При разработке табеля необходимо включить в функционал цветовые идентификаторы над значениями полей вывода чисел текущего месяца для обозначения причин неявок на рабочее место. Вся информация поступаемая в табель учета берется из формы отчета о проделанных работах сотрудника. Вывод ФИО сотрудников группируется по отделам , по ним же и происходит фильтрация.

<h5>Учет выработки</h5>

>Учет выработки представляет собой совокупность способов подсчета выработки, и документирования данных о ней. Учет на каждом предприятии организовывается в зависимости от характера производства, технологических условий, а также организации труда.

Пример официального документа учета выработки показан ниже.

<img src="/images/mainTable/reportWr.png" width="100%" height="auto"><br>

При разработке системы учета выработки сдельщиков необходимо учитывать следующие значения:
---Список ФИО сотрудников(включая фильтрацию по отделам) 
При переходе на конкретный сегмент, отображается:
---ФИО выбранного сотрудника
---Должность выбранного сотрудника
---Вид работ(список присвоенных видов работ)
---Вывод всех чисел текущего месяца - данные поля заполняются указателем о выполненных видах работ (шт.) за текущий день (возможное решение о переносе отработанных часов сотрудника-сдельщика из общего табеля учета рабочего времени в табель учета выработки)
